# Visitor Pass Management System (MERN)

A full-stack application to manage visitor entries, issue digital QR passes, and track check-in/check-out times.

## Features
- **Authentication:** Admin/Staff login.
- **Visitor Management:** Add visitors, generate QR codes.
- **Dashboard:** Real-time status tracking and statistics.
- **Digital Pass:** Generates a printable pass with QR code.

## Tech Stack
- **Frontend:** React (Vite), Bootstrap, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Tools:** JWT for Auth, React-QR-Code
